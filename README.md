# Custom Ping Utility Using ICMP

A simplified implementation of the ping utility using raw sockets and ICMP Echo Request/Reply packets.

## Requirements

- Python 3.x
- Administrator/root privileges

## Running

```bash
python ICMP.py

Enter the destination host when prompted.
